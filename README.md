# Data Matrix Quality Scanner

[![Build Windows](https://github.com/YOUR_USERNAME/datamatrix-quality-scanner/actions/workflows/build.yml/badge.svg)](https://github.com/YOUR_USERNAME/datamatrix-quality-scanner/actions)

Программа для оценки качества печати двумерных штриховых кодов Data Matrix по стандарту **ГОСТ Р 57302-2016** (ISO/IEC 15415:2011).

## Возможности

- 📊 Оценка по 8 параметрам качества ГОСТ Р 57302-2016
- 📷 Поддержка камеры и загрузки файлов
- 📄 Экспорт отчетов (TXT, JSON)
- 🎨 Современный графический интерфейс
- 🏭 Готовые сборки для Windows x64 и x86

## Параметры оценки

| Параметр | Обозначение | Описание |
|----------|-------------|----------|
| Декодирование | Decode | Успешность чтения кода |
| Контраст символа | SC | Разница между светлыми и темными элементами |
| Модуляция | MOD | Отношение контраста к общему диапазону |
| Запас по отражению | RM | Запас относительно порога |
| Повреждение шаблона | FPD | Целостность L-образного шаблона |
| Осевая неоднородность | AN | Равномерность по осям X и Y |
| Неоднородность сетки | GN | Регулярность расположения модулей |
| Неисп. коррекция ошибок | UEC | Запас коррекции Reed-Solomon |
